# ⏱️ Stopwatch App

A simple and modern **Stopwatch Web App** built using **HTML, CSS, and JavaScript**.  
This project demonstrates time-based logic, DOM manipulation, and responsive UI design with interactive features.

## ✨ Features

* ▶️ Start Timer  
* ⏸️ Pause & Resume (same button)  
* 📝 Auto Lap Capture on Pause  
* 🔄 Reset Timer  
* 🌗 Dark / Light Mode Toggle  
* ⏱️ Real-time time tracking  
* 📱 Fully Responsive Design  
* 🎨 Clean Minimal UI
